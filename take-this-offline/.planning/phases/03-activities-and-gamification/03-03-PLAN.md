- Shuffle distractors + correct answer using a stable shuffle seeded on word.id (use a simple Fisher-Yates shuffle; Math.random() is fine since content correctness does not depend on seed):
```typescript
const options = useMemo(() => {
  const arr = [word.definition, ...distractors]
  // Fisher-Yates shuffle
  for (let i = arr.length - 1; i > 0; i--) {
    const j = Math.floor(Math.random() * (i + 1));
    [arr[i], arr[j]] = [arr[j], arr[i]]
  }
  return arr
}, [word.id, distractors])
```
